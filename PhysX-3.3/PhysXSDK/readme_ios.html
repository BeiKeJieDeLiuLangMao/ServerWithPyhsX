NVIDIA® PhysX® SDK for iOS ReadMe

Location of Binaries:

  • SDK static libraries: Lib/ios and Lib/ios64
  • Snippet executables: Bin/ios and Bin/ios64 (not pre-built as app needs signature in order to run)

Building SDK Source with XCode:

  • XCode projects are in Source/compiler/xcode_ios | xcode_ios64
  • Source Distribution: Open PhysX.xcodeproj to build the entire SDK source within XCode
  • Binary Distribution: Open PhysXExtensions.xcodeproj, PhysXCharacterKinematic.xcodeproj or PhysXVehicle.xcodeproj to build the available SDK source within XCode
  • Build from command line using e.g.: xcodebuild -project PhysX.xcodeproj -sdk iphoneos -alltargets -configuration (debug|checked|profile|release)

Building and Running Snippets with XCode:

  • Open XCode project Snippets/compiler/xcode_ios|xcode_ios64/Snippets.xcodeproj
  • Select one of the snippet schemes from the drop down list (e.g. HelloWorld)
  • Make sure the iPad is selected as opposed to the simulator
  • Press "Run" to build and run the snippet
  • In order to build and run release mode you need to select "Edit Scheme..." and change the "Build Configuration" to "release". Additionally, after building, you need to manually choose the "Executable" under ".../Bin/ios/SNIPPET_EXE_NAME". See "Known XCode issues" below.

Notes:

  • No support for building the SDK or Snippets with makefiles currently
  • No rendering support for Snippets
  • Current xcode projects target:
    • XCode version: 6.3
    • Apple clang version 6.1
    • OS: iPhoneOS 8.3
    • platform: armv7|arm64
  • Building for the simulator is currently not supported
  • Known XCode issues:
    • Differently named products across configurations don't seem to be supported properly. Changing the active configuration (Edit Scheme... , Run , Build Configuration) doesn't display the correct Executable products. When running the configuration the right executable is picked however.

Limitations:

  • The maximum number of shapes in each scene must be less than 32767.
  • The maximum number of dynamic actors added to each scene must be less than 65535.
  • The maximum number of broadphase overlap pairs must be less than 65535. Overlap pairs above this limit will be neglected.
